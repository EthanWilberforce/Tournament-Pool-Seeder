How to use PoolSeeder.me
   Enter entrant tags separated by commas (ex/ M2K, Mango, Leffen, Ethan etc) listed in order of seed priority in the text field    below. Then click the button!

   Restrictions:

  • Use only letters and numbers, no special characters (ex/ no $,#,|,)
  • Make sure you have no leading, trailing, or double commas.

   How it works
   The script first adds the first name on the list to the first position on the first graph, then adds the second name to the first    position on the second graph, and so on. This process makes sure that everyone who is supposed to get out of pools likely will    proceed to bracket.

Pool Seeder

Enter entrant tags separated by commas (ex/ M2K, Mango, Leffen, Ethan etc), listed in order of seed priority.

How does the seeder place names?

The script first adds the first name on the list to the first position on the first graph, then adds the second name to the first position on the third graph, and so on. This process makes sure that everyone who is supposed to get out of pools, likely will.
